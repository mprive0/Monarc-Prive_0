# ============================================================
# Monarc Privé — Deployment Guide
# ============================================================
#
# FASTEST PATH TO PRODUCTION TODAY:
#
# ─────────────────────────────────────────────────────────────
# STEP 1: SUPABASE (10 minutes)
# ─────────────────────────────────────────────────────────────
# 1. Go to supabase.com → New Project
# 2. Name: monarc-prive | Region: US West (closest to Scottsdale)
# 3. In SQL Editor: paste + run supabase/schema.sql
# 4. In Settings > API: copy your URL, anon key, service key
# 5. In Authentication: enable Email, Google, Apple providers
#
# ─────────────────────────────────────────────────────────────
# STEP 2: ANTHROPIC API KEY (2 minutes)
# ─────────────────────────────────────────────────────────────
# 1. Go to console.anthropic.com
# 2. API Keys → Create new key
# 3. Copy into ANTHROPIC_API_KEY above
#
# ─────────────────────────────────────────────────────────────
# STEP 3: STRIPE (15 minutes)
# ─────────────────────────────────────────────────────────────
# 1. Go to dashboard.stripe.com
# 2. Developers → API Keys → copy live keys
# 3. Create webhook: endpoint URL = https://your-backend.com/api/webhooks/stripe
#    Events to listen: payment_intent.succeeded, payment_intent.failed,
#    customer.created, charge.dispute.created
#
# ─────────────────────────────────────────────────────────────
# STEP 4: SENDGRID EMAIL (5 minutes)
# ─────────────────────────────────────────────────────────────
# 1. app.sendgrid.com → Settings → API Keys → Create API Key
# 2. Full access
# 3. Verify sender domain (monarcprive.com)
#
# ─────────────────────────────────────────────────────────────
# STEP 5: DEPLOY BACKEND (10 minutes with Render)
# ─────────────────────────────────────────────────────────────
# 1. Push your code to GitHub
# 2. render.com → New → Web Service → connect GitHub repo
# 3. Root directory: server/
# 4. Build command: npm install && npm run build
# 5. Start command: node dist/index.js
# 6. Add all environment variables from this file
# 7. Deploy — you get a URL like: https://monarc-api.onrender.com
#
# ─────────────────────────────────────────────────────────────
# STEP 6: DEPLOY FRONTEND (5 minutes with Vercel)
# ─────────────────────────────────────────────────────────────
# 1. vercel.com → New Project → Import from GitHub
# 2. Framework: Vite (React)
# 3. Add env vars: VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY,
#    VITE_STRIPE_PUBLISHABLE_KEY, VITE_API_URL (your Render URL)
# 4. Deploy — you get monarcprive.vercel.app (then add custom domain)
#
# ─────────────────────────────────────────────────────────────
# STEP 7: DOMAIN (optional, 30 min with DNS propagation)
# ─────────────────────────────────────────────────────────────
# 1. Buy monarcprive.com on Namecheap or GoDaddy (~$15/yr)
# 2. In Vercel: Settings → Domains → Add monarcprive.com
# 3. Update DNS records at your registrar as instructed
#
